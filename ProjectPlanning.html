Project planning

Why Plan your projects

One of the hardest parts of a project is keeping the end goal in mind. One of the best ways to stay focused and motivated is to have a well planned project. To keep a project running smoothly I like to use khanban boards, github, and some form of group communication platform like, microsoft teams or slack.

Tools

Currently I am using teams to communicate with my ballot app team.

I am using github projects for my khanban board.

For my repo I am using github.com

Alternatives to github khanban

This is Gitkraken. For me I think this is the best khanban board I have used. I say this because the tooling is the nicest to use.

This is Trello. It is another free khanban board. It was my first intro to Khanban boards. I feel that it is a well made program with simple tools. Personaly I like it more than Githubs khanban Board

So if I like these other boards better why did we choose githubs khanban? THe annswer is that it was the easiest to access. It is in a shared location. The most important reason though is this is what the team chose to use.

Alternatives to Teams

This is Slack. Personaly I dont care for slack. I never felt all that comfertable using it. Another thing to be consered about is that it will only save the last 1000 messages.

This is Discord. It is another free communications app.It is poppular among people playing games. I have not used it often.

So Why did we choose Microsoft Teams? It again comes down to availability. We all were already using teams to communicate with each other. It allows us to see past conversations.

Take aways

There are a few important things to remember when coosing the tools for a project. Ask yourself, Who will be using it, Why will you use it and can everyone use it. Once a choice of tools has been selected as an individual on a team make sure you use them. If you don't you will cause your team more stress and be a less effective member of the team.
