At the start of my journy learning Linux I didn't know what I wa trying to learn. I knew that linux was an OS similar to Windows or Mac. Other than this I only had heard that it was what gave the user the most control over the computer. So to learn what linux is and how to use it i made a list of goals for myself as seen above.

Installing Linux

To learn Linux you need to have access to linux. I can think of three ways to get linux on your computer. YOu can install a linux distro on your computer( probably the hardest way), You can use adocker container(probably the easiest but harder to come back to and start at the same point again) and the way I am currently doing it with A virtual machine.(I am using Kali Linux through the windows store.)

File Traversal

How do I see what Files I have in a directory?

ls This is the command to see the files in a folder

How do I change to a different directory?

cd This is the command to change directories. In the previous example you see that I have a folder called hacktools. To move into hacktools I will type "cd hacktools". You can see that you are now in /hacktools but it doesnt show you whats in hacktools. Ask yourself how can I see whats in hacktools?

How do I go back in the directory?

cd .. This command lets me go back one in the file directory.

How do I make a new directory?

mkdir nameOfDirectory This command lets me create a new file directory at the current location. After you create the directory you will need to either ls to see if it was made or cd to move into it.

How do I remove directory?

rmdir nameOfDirectory This is the command to remove a directory.

Bonus

--help This is the command to learn more about ls. It opens up the linux manual with all the little command that go with ls. It can be used on any of the commands that we tried and others. It is how you will become a power user in Linux.
